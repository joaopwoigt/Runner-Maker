# Runner Maker

App web de planos de corrida personalizados — gerador de planos periodizados, dashboards de acompanhamento e persistência em nuvem (Supabase).

Arquivo único `index.html` (HTML + CSS + JS, sem build). Hospedado via GitHub Pages.

**Acesso:** abra a URL do GitHub Pages no navegador (desktop ou celular). Cadastre-se com e-mail/senha para gerar seu plano.
